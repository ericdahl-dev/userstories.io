<section class="section">
  <div class="container narrow">
    <div class="feature-grid">
      <article class="feature-block">
        <h2>For developers</h2>
        <ul>
          <li>GitHub OAuth sign-in with encrypted token storage</li>
          <li>Projects linked to any repo you own</li>
          <li>Shareable portal links (rotatable to revoke access)</li>
          <li>Triage inbox across all projects</li>
          <li>One-click accept → GitHub issue creation with backlink</li>
          <li>Dismiss noise without polluting your issue tracker</li>
          <li>Manual or synced <strong>shipped</strong> status when issues close</li>
        </ul>
      </article>
      <article class="feature-block">
        <h2>For collaborators</h2>
        <ul>
          <li>No GitHub account required</li>
          <li>Passwordless magic-link authentication</li>
          <li>Persistent sessions across visits</li>
          <li>Submission history with live status (<code>pending</code> → <code>accepted</code> → <code>shipped</code>)</li>
          <li>AI-assisted story refinement grounded in project history and repo context</li>
        </ul>
      </article>
    </div>
  </div>
</section>
